特性

Features

⚡

模式系统

6 种内置模式，每种模式切换性格、工具、UI、权限。支持自定义 YAML 模式。

Mode System

6 built-in modes, each switching personality, tools, UI, and permissions. Custom YAML modes supported.

🔪

Dr. Sharp

内置人生教练人格，三阶段心理分析工作流。用手术刀般精准的方式审视你的代码和决策。

Dr. Sharp

Built-in life coach persona with a 3-stage psychological analysis workflow.

📋

代码问责

AI 修改代码后强制展示 diff 并追问修改理由，记录决策日志。

Code Accountability

Force-show diff after AI edits, ask for modification reasons, log decisions.

🔌

多模型供应商

支持 Anthropic/OpenAI/Gemini/Grok 等 7 个 API 提供商。

Multi-API Providers

Support 7 API providers: Anthropic, OpenAI, Gemini, Grok, Bedrock, Vertex, Foundry.

🖥️

Computer Use

屏幕截图、键鼠控制。Chrome 浏览器自动化、表单填写、数据抓取。

Computer Use

Screen capture, keyboard/mouse control. Chrome browser automation and data scraping.

💰

穷鬼模式

省 Token 模式，关闭记忆提取和键入建议，显著减少消耗。

Budget Mode

Token-saving mode — disable memory extraction and input suggestions to reduce costs.
